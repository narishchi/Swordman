# Swordman Game

## รายละเอียดโปรเจค
โปรเจคนี้เป็นเกม **Swordman** ที่สร้างขึ้นโดยใช้ **Unity**  
- การเดิน (player_move)
- การกระโดด (player_jump)
- การบาดเจ็บ (Hurt)
- การตาย (Death)
