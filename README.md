# Swordman Game

## รายละเอียดโปรเจค
โปรเจคนี้เป็นเกม **Swordman** ที่สร้างขึ้นโดยใช้ **Unity**  
- ระบบการเดิน (Move)
- ระบบการกระโดด (Jump)
- ระบบการต่อสู้ (Attack)
- ระบบการตาย (Death) และ Game Over
- ระบบการรีเซ็ตเกม (Restart)
